# 详谈·左晖：做难而正确的事

**受访者**：左晖（链家、贝壳找房、自如创始人）
**采访者**：李翔
**出版**：新星出版社《详谈》系列01
**访谈时间**：2020年9月（主访）+ 2017年（补录）

> 本书核心：左晖用20年时间在中国最复杂、最被轻视的行业之一——二手房中介——建立了一家真正改变行业规则的公司。他的方法论不是颠覆，而是从最基础的地方做起：真房源、不吃差价、让服务者有尊严。

---

## 人物背景

| 项目 | 内容 |
|------|------|
| 创办链家 | 2001年，时年30岁 |
| 核心品牌 | 链家（经纪）、贝壳找房（平台）、自如（长租） |
| 贝壳上市 | 2020年8月，纽交所，市值超700亿美元 |
| 核心理念 | 做难而正确的事；有尊严的服务者；更美好的居住 |
| 行业特点 | 重线下、重信任、重数据，慢公司 |

---

## 核心金句
