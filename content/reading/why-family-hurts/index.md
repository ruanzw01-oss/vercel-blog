# 《为何家会伤人》精读笔记

> 作者：武志红
> 版本：升级版

家是我们最初学习爱与被爱的地方，也是很多人心理创伤的来源。武志红用心理咨询案例，拆解中国式家庭关系中最常见的伤害模式——溺爱、焦虑转嫁、孩子化父母、权力规则、错误的爱的逻辑。

这本书的核心命题是：**家之所以会伤人，是因为爱的方式出了问题，而不是爱的意图。** 大多数父母都爱孩子，但很多人用「爱」的名义做的事，实际上是在满足自己，而不是在真正回应孩子的需要。

---

| 章节 | 核心问题 |
|------|----------|
| 溺爱的心理真相 | 溺爱本质是父母对自己「内在小孩」的满足，不是对孩子的真爱 |
| 焦虑的转嫁 | 父母把自己的焦虑和压力转嫁给孩子，让孩子承担本不该承担的重量 |
| 孩子化的父母 | 父母在情感上像孩子，反过来要求子女照顾自己 |
| 夫妻沟通 | 揣测、评价、出主意，这三种「爱」的方式如何破坏关系 |
| 权力规则带回家 | 职场的权力逻辑进入家庭，摧毁夫妻和亲子关系 |
| 分离 | 每一次分离都是成长的机会，也是最容易被父母阻断的时刻 |
| 爱的谎言 | 「没有父母不爱孩子」「我爱你所以你要听我的」——这些话背后的真相 |
