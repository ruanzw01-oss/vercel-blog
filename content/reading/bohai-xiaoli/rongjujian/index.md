# 荣枯鉴

> 相传为五代十国冯道所著，与《全经》《观止经》并称「官场三部曲」收官之作。
> 善恶，贵贱，名实，福祸，此经道尽人间荣枯。

---

## 关于这本书

《荣枯鉴》是一部讲透官场底层规律的古典典籍，以九句核心格言为骨架，通过大量历史案例，剖析善恶、名利、富贵、祸福、荣枯的本质逻辑。

| 格言 | 核心主题 |
|------|---------|
| 善恶有名，智者不居 | 祛除道德幻觉，看清善恶的权力本质 |
| 惜名者伤其名，惜身者全其身 | 名是毛，实是皮；惜身是大智慧 |
| 名利无咎，竹之非罪 | 名利是工具，错在人的方式与尺度 |
| 富贵有常，其道乃实 | 富贵有方法论：务实、历实、踏实 |
| 福祸非命，其道乃察 | 福祸不是命运：察人、察势、察己 |
| 无忧则患烈 | 忧患意识是生存基础 |
| 忧国者失身，有己者安命 | 保全自身是长期主义 |
| 荣获为君子，枯必为小人 | 环境塑造人性，根本是让人变荣 |
| 上不离心，非小人难为 | 官场权力链条的运作逻辑 |
| 道义失之无成 | 祛除「正义必胜」的幻想 |
