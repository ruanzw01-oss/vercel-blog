# 《随机漫步的傻瓜》

**作者**：纳西姆·尼古拉斯·塔勒布（Nassim Nicholas Taleb）
**出版**：2001年（中译本：中信出版集团，2012年）
**系列**：Incerto 五部曲之一

> 本书核心：我们习惯于把运气误认为能力，把随机现象误认为规律。市场、人生、历史中充斥着被随机性愚弄的傻瓜——而我们自己往往正是其中之一。

---

## 核心主题

| 主题 | 核心观点 |
|------|----------|
| 另类历史 | 我们只看到实现的历史，看不到未实现的无数可能路径 |
| 存活者偏差 | 成功者可见，失败者隐身，导致我们系统性高估成功的可重复性 |
| 黑天鹅 | 稀有的极端事件被严重低估，却往往决定命运 |
| 偏态与期望值 | 高频小赢 vs 低频大赔，概率不等于期望值 |
| 归纳法陷阱 | 过去的规律无法保证未来，火鸡在感恩节前的逻辑 |
| 概率盲 | 人类大脑天生不擅长处理概率，专家也不例外 |
| 随机性与情绪 | 血清素、社会地位感知让我们对随机结果过度反应 |
