# 《商道》阶段整理

作者：【韩】崔仁浩

这本书不是普通的商业传奇，而是借现代企业家金起燮之死，追索朝鲜巨商林尚沃的一生，并把“赚钱的技巧”推进到“如何立身、如何用财、如何对人”的层面。

整理分为 7 篇（概览 + 5 个阶段 + 结语）：每篇采用“情节梳理 + 商业逻辑 + 人物与价值观解读”的方式，尽量保留原书的历史纵深和现代呼应。另附 `yuanwen` 目录，按阶段分段保存，方便后续继续扩写。
