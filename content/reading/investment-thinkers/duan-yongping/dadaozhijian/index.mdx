# 大道至简的财富逻辑

import { Cards } from 'nextra/components'

> 做对的事情，把事情做对。—— 段永平

**《段永平投资问答录：大道至简的财富逻辑》** 是马小奔所著的第三方解读作品（2026年出版），系统梳理了段永平的投资哲学与商业智慧。全书以段永平在雪球、博客等平台的公开问答为素材，分四章四十节深度解析其思想体系。

本书并非段永平本人撰写，而是作者对段永平投资理念的分析与阐释，将原始问答中的核心观点提炼为结构化的思想框架。

---

## 全书结构

| 章节 | 主题 | 核心关键词 |
|------|------|------------|
| 第一章 | **本分的原点** — 投资中那些简单而极其重要的常识 | 本分、常识、确定性、慢即是快、停止清单 |
| 第二章 | **商业的本质** — 如何从纷繁复杂中一眼识别伟大公司 | 商业模式、护城河、差异化、企业文化、现金流 |
| 第三章 | **无为的定力** — 在剧烈波动中保持理性的持股哲学 | 估值、持有、仓位、杠杆、时间 |
| 第四章 | **实战的复盘** — 从网易到苹果的十年百倍财富密码 | 网易、苹果、茅台、腾讯、巴菲特、本分传承 |
