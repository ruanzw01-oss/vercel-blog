# 借命而生

> **石一枫 著** | 2018年出版 | 一部关于命运、正义与人心的当代长篇小说

---

## 关于这本书

《借命而生》是70后作家石一枫的长篇小说，讲述了一个跨越近二十年的追逃故事。

看守所管教杜湘东，一个心怀刑侦梦却被分配到郊县看守所的警校毕业生，与两个年轻的盗窃犯——许文革和姚斌彬——命运纠缠了大半生。两次越狱、一场漫长的追捕，三个人的人生轨迹在时代洪流中不断交错，最终在世纪之交的夜晚完成了各自的救赎。

> 小说标题"借命而生"，既是姚斌彬把自己的命托付给许文革、让他替自己活下去的壮烈选择，也是每个人都在向他人、向时代借一口气活着的隐喻。

## 核心人物

| 人物 | 身份 | 关键词 |
|------|------|--------|
| 杜湘东 | 看守所管教（警察） | 理想主义、憋闷、执念、正义感 |
| 许文革 | 盗窃犯 / 越狱者 | 硬气、重情、技术天才、亡命天涯 |
| 姚斌彬 | 盗窃犯（许文革同案犯） | 软弱、善良、自我牺牲 |
| 刘芬芳 | 杜湘东妻子 | 忧愁、文艺、现实 |

---

## 章节目录

全书共十七章，以杜湘东的视角为主线，穿插许文革的逃亡与重逢。

| 章节 | 大致内容 |
|------|----------|
| 第一至三章 | 杜湘东的看守所生活，许文革与姚斌彬入所，第一次越狱 |
| 第四至六章 | 追捕与落网，姚斌彬之死，杜湘东的执念 |
| 第七至九章 | 时代变迁，杜湘东的婚姻与困境，许文革的第二次越狱 |
| 第十至十二章 | 漫长追踪，许文革的隐匿生活，杜湘东的坚守 |
| 第十三至十五章 | 线索浮现，真相逐渐清晰，命运再次交汇 |
| 第十六至十七章 | 最终对决与和解，"借命而生"的真正含义揭晓 |
