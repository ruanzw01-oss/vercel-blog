
# 量化笔记：双动量策略（Dual Momentum）深度解析

### 1. 核心定义

双动量策略由盖瑞·安东纳奇（Gary Antonacci）提出，其核心在于：不仅要选出**表现最好**的资产，还要确保该资产正处于**上升趋势**中。

> **公式总结**：
> 
> **双动量 = 相对动量 (Relative) + 绝对动量 (Absolute)**

---

### 2. 两大维度拆解

|**维度**|**解决的问题**|**逻辑表述**|**核心作用**|
|---|---|---|---|
|**相对动量**|**买什么？**|在资产 A、B、C 中，谁过去 N 个月涨得最快？|**选股（寻找 Alpha）**：捕捉最强轮动板块。|
|**绝对动量**|**买不买？**|选出的资产过去 N 个月的收益是否 > 0（或跑赢现金）？|**生存（控制回撤）**：识别风险，决定是否空仓。|

---

### 3. 策略逻辑流图（GEM 模型变种）

1. **步骤一：相对筛选**
    
    - 对标的池（如：沪深300、纳指、黄金）计算过去 12 个月的收益率。
        
    - 提取排名靠前的 Top N 标的。
        
2. **步骤二：绝对过滤**
    
    - 检查 Top N 标的的收益率是否为正。
        
    - **IF 收益 > 0**：按比例买入。
        
    - **IF 收益 < 0**：全仓切换至避险资产（如国债、现金、黄金）。

### 4. 生产级 Python 伪代码架构

Python

```
# 计算相对排名
rank_list = calculate_relative_momentum(assets, lookback=252)
top_assets = rank_list.nlargest(2)

# 执行绝对择时过滤
final_targets = []
for asset in top_assets:
    # 只有当收益为正，且跑赢大盘/现金时才入场
    if momentum_score[asset] > 0:
        final_targets.append(asset)

# 执行调仓
if not final_targets:
    go_to_cash_or_bond() # 生存：空仓或买债
else:
    rebalance(final_targets) # 选股：满仓最强标的
```

### 5. 常见误区纠偏

- **❌ 误区**：双动量是指长短周期结合（如 20日+60日）。
    
- **✅ 真相**：那叫“多频率动量”。双动量特指“横向对比”与“纵向择时”的结合。
    
- **❌ 误区**：相对动量最强就一定要买。
    
- **✅ 真相**：如果绝对动量为负（全市场暴跌），即便相对第一也要空仓，这是生存的关键。
    

---

### 6. 实盘优化建议

1. **调仓频率**：建议按月或按双周，避免每日调仓的高额手续费。
    
2. **调仓死区**：引入 2% 的缓冲区，比例微调时不操作，减少碎股交易。
    
3. **避险切换**：当触发绝对动量卖出时，可以切换至国债 ETF，利用空仓期赚取稳健利息。
