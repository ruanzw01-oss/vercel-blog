# 动量因子策略 vs 追涨杀跌：看似相同，本质天差地别

很多人第一次听到"动量策略"时会问：**这不就是追涨杀跌吗？** 表面上看，两者都是"买涨的、卖跌的"，但背后的逻辑、执行方式和最终结果截然不同。

---

## 1. 先搞清楚两个概念

**动量因子策略**，简单说就是：过去一段时间涨得好的资产，接下来大概率还会继续涨。这是经过几十年学术研究验证的统计规律，属于量化投资的核心因子之一。

**追涨杀跌**，说白了就是：看到涨了就忍不住买，看到跌了就慌忙卖。这是一种典型的散户行为偏差，本质上是被情绪牵着鼻子走。

| | **动量因子策略** | **追涨杀跌** |
|---|---|---|
| **本质** | 量化因子，基于统计规律 | 行为偏差，基于情绪冲动 |
| **理论来源** | 学术研究（1993 年被正式提出） | 行为金融学中的"羊群效应" |
| **决策依据** | 过去 3-12 个月的收益率排名 | "感觉还会涨" / "怕再跌" |
| **触发方式** | 预设规则，自动执行 | 临时起意，拍脑袋决定 |

---

## 2. 五个关键区别

### 第一：有没有交易系统

- **动量策略**：有一套完整的规则——回看多长时间、多久调一次仓、什么时候止损、仓位怎么分配，全部提前定好。
- **追涨杀跌**：没有任何系统。涨了就追，跌了就跑，全凭当时的感觉。

### 第二：看的是"趋势"还是"今天的涨跌"

- **动量策略**：看的是中长期趋势——过去半年到一年表现强势的资产，统计上倾向于继续走强。
- **追涨杀跌**：盯的是短期波动——今天涨停了赶紧追，明天跌了赶紧卖。

### 第三：有没有退出机制

- **动量策略**：内置"绝对动量"过滤。当整个市场都在跌的时候，哪怕某个资产相对排名第一，也会主动切换到债券或现金避险。
- **追涨杀跌**：没有退出机制。往往在最高点冲进去，在最低点恐慌割肉。

### 第四：交易频率

- **动量策略**：通常按月或按季度调仓，操作频率低，交易成本可控。
- **追涨杀跌**：频繁买卖，一天可能操作好几次，手续费和滑点不知不觉就把利润吃掉了。

### 第五：情绪参与程度

- **动量策略**：规则说了算，涨了不贪、跌了不慌，情绪靠边站。
- **追涨杀跌**：情绪说了算。贪婪的时候满仓梭哈，恐惧的时候全部清仓。

---

## 3. 为什么容易搞混？（相似之处）

说实话，这两者确实有不少表面上的相似点：

| **相似点** | **具体表现** |
|---|---|
| 都是"买强卖弱" | 都倾向于买入上涨的资产、卖出下跌的资产 |
| 都是顺势而为 | 都不是逆势抄底，而是跟着市场方向走 |
| 牛市里看起来一模一样 | 行情好的时候，两者的操作可能完全一致 |
| 趋势反转时都会亏 | 当市场突然掉头，两者都会面临回撤 |

> **但关键区别在于**：动量策略在"看起来一样"的操作背后，有严格的风控和退出机制；追涨杀跌则完全没有。

---

## 4. 举个例子就明白了

假设沪深300连续上涨了 6 个月：

**动量策略怎么做：**
1. 月末按规则计算过去 12 个月收益率排名
2. 沪深300排名靠前 → 按规则纳入持仓
3. 检查绝对动量为正 → 确认继续持有
4. 下个月重复同样的步骤
5. 如果排名下滑或绝对动量转负 → 按规则切换到避险资产

**追涨杀跌怎么做：**
1. 看到沪深300涨了好几天 → "肯定还会涨！"
2. 一把梭哈满仓买入
3. 第二天跌了 2% → "完了完了要崩了"
4. 赶紧割肉全卖了
5. 第三天又涨回来了 → "早知道不卖..."

---

## 5. 学术研究怎么说

动量效应是金融学里最经得起检验的规律之一：

- **1993 年**，两位学者首次系统性地证明：买入过去 3-12 个月的强势股、卖出弱势股，能获得显著的超额收益。
- **2013 年**，研究进一步发现：动量效应不仅存在于股票市场，在债券、外汇、商品等多个市场中都普遍存在。
- **1997 年**，动量因子被正式纳入资产定价模型，成为解释股票收益的重要因子之一。

而"追涨杀跌"在学术研究中被明确归类为**投资者行为偏差**，是导致散户长期跑输市场的主要原因之一。

---

## 6. 一张表总结

| | **动量因子策略** | **追涨杀跌** |
|---|---|---|
| 有没有系统 | 有完整的交易规则 | 没有，全凭感觉 |
| 谁在做决策 | 规则做决策 | 情绪做决策 |
| 风控机制 | 有绝对动量过滤 | 没有任何风控 |
| 调仓纪律 | 固定频率，按计划执行 | 想买就买，想卖就卖 |
| 长期收益预期 | 正期望，有超额收益 | 负期望（手续费 + 追高割肉） |
| 学术支撑 | 几十年的实证研究 | 被归类为行为偏差 |

> 一句话总结：**动量策略是用规则捕捉趋势，追涨杀跌是用情绪追逐价格。** 前者是纪律，后者是冲动。
