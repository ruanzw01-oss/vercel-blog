# 职业发展

## 关于职业选择

做技术还是做管理？这是很多人都会面临的问题。

### 技术路线

```
初级工程师 → 中级工程师 → 高级工程师 → 技术专家 → 架构师
```

**适合**：热爱技术、享受解决问题的人

### 管理路线

```
工程师 → 技术组长 → 技术经理 → 技术总监 → CTO
```

**适合**：善于沟通协调、喜欢带团队的人

## 持续学习

在技术行业，学习是一辈子的事：

- 📚 每天留出 30 分钟学习时间
- 🎯 聚焦一个方向，深耕细作
- 🔄 定期回顾，知识沉淀
- ✍️ 输出倒逼输入

## 软技能

技术能力决定下限，软技能决定上限：

| 技能 | 重要性 | 如何提升 |
|------|--------|---------|
| 沟通表达 | ⭐⭐⭐⭐⭐ | 多写文档、多做分享 |
| 项目管理 | ⭐⭐⭐⭐ | 实践 + 反思 |
| 解决问题 | ⭐⭐⭐⭐⭐ | 刻意练习 |
| 团队协作 | ⭐⭐⭐⭐ | 开放心态、主动配合 |

---

> 💡 职业发展是长跑，保持节奏最重要。
